# java-chess

- 체스판을 초기화한다.
    - 말의 위치 값은 가로 위치는 왼쪽부터 a ~ h이고, 세로는 아래부터 위로 1 ~ 8로 구현한다.
    - 각 진영은 검은색(대문자)과 흰색(소문자) 편으로 구분한다.

```shell script
RNBQKBNR  8 (rank 8)
PPPPPPPP  7
........  6
........  5
........  4
........  3
pppppppp  2
rnbqkbnr  1 (rank 1)

abcdefgh
```
## 기능 요구사항

- [x] 빈 판을 생성한다.
    - [x] 체스말들을 생성한다.
    - [x] 체스말들을 초기 위치에 놓는다.
    - [ ] 콘솔 출력
